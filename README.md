Neighborly is a <b>mobile-first web application</b> that helps neighbors connect, share skills, organize events, and build stronger communities.


## ✨ Features

- 🏠 <b>Home Feed:</b> Find help, offer help, see local events and announcements.
- 🛠️ <b>Skills Marketplace:</b> Search and connect with local skill providers.
- 🎉 <b>Events:</b> Discover and join community events.
- 👥 <b>Groups:</b> Join and manage community groups.
- 📱 <b>Mobile-optimized UI</b> with bottom navigation and beautiful gradients.

## 🚀 Getting Started

```sh
# 1. Install dependencies
npm install

# 2. Start the development server
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.
